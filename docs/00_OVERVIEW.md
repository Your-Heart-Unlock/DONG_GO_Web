# Project Overview

## 프로젝트 목적
네이버 지도에 개인적으로 저장해 둔 맛집 데이터를 기반으로,
Firebase에 데이터를 저장하고 웹 서비스로 제공한다.

이 서비스는 공개 플랫폼이 아니라,
**나와 내 주변 사람들이 함께 사용하는 폐쇄형 맛집 공유 서비스**다.

## 왜 필요한가
- 친구들이 반복적으로 “여기 어때?”, “이 동네 맛집 있어?”라고 물어봄
- 말로 설명하거나 매번 링크 보내는 것이 비효율적
- 우리끼리 신뢰 가능한 맛집 풀(pool)을 만들고 싶음

## 핵심 컨셉
- 지도 기반
- 리뷰 커뮤니티 ❌
- SNS ❌
- **집단 큐레이션 메모장 ⭕**

## MVP 범위
포함:
- 가게 추가 (네이버 지도 기반)
- 방문 기록 / 리뷰 / 평가
- 링크 공유
- 관리자 승인 기반 운영

제외:
- 친구 추가 / 팔로우
- 공개 검색 노출
- 알고리즘 추천
