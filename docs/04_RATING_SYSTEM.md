# Rating System

## 저장 구조
- ratingTier: "S" | "A" | "B" | "C" | "F"
- DB에는 tier만 저장
- UI 문구는 별도 매핑

## 기본 라벨 매핑

| Tier | Label | Description |
|----|------|-------------|
| S | 전파각 | 진짜 너무 좋아서 남들에게 추천하고 싶은 곳 |
| A | 동네강자 | 이 동네에서는 이 집 정도면 잘친다 |
| B | 평타 | 손해는 안 보는 무난 |
| C | 땜빵 | 굳이지만 급하면 가능 |
| F | 지뢰 | 추천 불가, 피하는 게 낫다 |

## 설계 원칙
- 카피는 Settings에서 교체 가능
- 정렬/집계는 tier 기준
