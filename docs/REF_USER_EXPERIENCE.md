# User Experience Flow

## 메인 화면
- 지도 + 검색
- 마커 클릭 → 하단 바텀시트
- 바텀시트 클릭 → 상세 페이지

## 상세 페이지
- place 기본 정보
- member 이상:
  - 리뷰 리스트
  - 사진
  - 방문 기록
- pending/guest:
  - 통계만 노출
  - 리뷰/사진/방문 맥락 숨김

## 가게 추가
- 네이버 지도 검색 기반
- placeId 기준 중복 방지
- 이미 존재하면 visit/review만 추가

## 리뷰/방문
- 방문 기록과 리뷰는 분리 개념
- 리뷰/방문은 작성자만 수정/삭제 가능

## UX 톤
- 개인 큐레이션
- 과한 소셜 요소 없음
- 친한 사람에게 추천하는 느낌
